1) Instalar cypress

npm install cypress o npm install cypress --save-dev

2) Una vez instalado cypress, abrir el motor desde la consola con el comando
     
     npx cypress open

3) Seleccionar la opcion E2E Testing

4)Seleccionar el buscador de Google Chrome
    Hacer click sobre el boton 'Start E2E Tesing in Chrome'

5) Abrir la carpeta TestFuncional.cy.js

6) Finalmente se ejecutaras los test de prueba.
